# Alumni SDN 1 Senteluk 2017

Website resmi alumni SDN 1 Senteluk angkatan 2017. Dibangun dengan HTML, CSS, dan JavaScript tanpa backend.

## Fitur Utama

- **Beranda** - Halaman utama dengan informasi lengkap
- **Agenda** - Kegiatan alumni yang dapat difilter berdasarkan tahun
- **Galeri** - Foto-foto kegiatan alumni
- **Alumni** - Daftar alumni dengan card grid
- **Admin Panel** - Kelola data alumni, agenda, dan galeri

## Cara Menjalankan

1. Clone repository ini
2. Buka `index.html` di browser
3. Untuk admin, akses `admin/login.html`
   - Username: admin
   - Password: senteluk2017

## Dark Mode

Website mendukung dark mode dan light mode. Toggle tersedia di navbar.
